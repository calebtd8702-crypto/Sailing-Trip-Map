# Maintenance log

Every job, every time. The pattern that saves boats: date · engine hours · what · parts used · next due. When something fails in a remote anchorage, this log is how you know what was touched last.

## Recurring schedule
| Interval | Job |
|----------|-----|
| Weekly (cruising) | Engine oil level, coolant, belts eyeball, bilge check, rig visual |
| 50 hrs | Racor / primary fuel filter check |
| 100 hrs | Engine oil + filter change |
| 250 hrs | Secondary fuel filter, impeller inspect |
| 6 months | Zincs (more often in marinas), seacock exercise, winch flush |
| Annual | Rig inspection aloft, liferaft/EPIRB/flare dates, haul-out or dive bottom job, service outboard |
| Before ocean passage | Full rig check, steering gear + autopilot inspect, storm sails aired, jackstay check |
